# Wordly: Word search Web Application

## Introduction

The wordly web application is a search dictionary application. It has features that allow users to search for words, definition, synonyms and store previous search.

## 
- Search for Word:  Allow users to input a word into an HTML form and fetch its definition from an external dictionary API.

- Display Definitions: It shows the word search definition, part of speech and example.

- History: Displays: Recently searched words and replaces it with current word searched 

- Word of the day: It allows user to get inspo by displaying random words fetched from a radom word API
